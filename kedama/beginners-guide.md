# 毛玉线圈物语新人指南

本指南旨在为新玩家提供一个简洁明了、能够快速上手毛玉线圈物语服务器( 毛线 ) 的毛线版本指南。喵窝版本请前往[喵窝指南](nyaa/beginners-guide.md)。

注意该指南可能会随着游戏设定而改变，欢迎玩家们在保持简单、简洁的原则上完善本页面。

## 关于 Minecraft

NyaaWiki **并不是** 针对《Minecraft》游戏的指南和知识库，它是针对 NyaaCat 社区和喵窝/毛线服务器的指南、手册和知识库系统。如果希望获得关于 Minecraft 游戏本身的信息，请访问：

* [中文 Minecraft Wiki](https://zh.minecraft.wiki)
* [英文 Minecraft Wiki](https://minecraft.wiki)

如果你是第一次玩 Minecraft，请首先参考官方 Wiki 的[《新手手册》](https://zh.minecraft.wiki/Tutorial%3A%E6%96%B0%E6%89%8B%E6%89%8B%E5%86%8C)。  
如果你之前没有在其它正版 Java 版服务器游玩的经历，请务必确认你使用的是 [Java 版客户端](https://zh.minecraft.wiki/Java%E7%89%88)，并通过**已购买 Minecraft Java 版的微软账号**登录游戏。
[网易（中国）版](https://mc.163.com/) 不受支持。

## 入服准备

### 加入须知

毛线物语服务器是公益、非营利、原版生存服务器。本服务器不设置生存玩家和管理员以外的权限，**不会索要任何个人隐私、金钱或物品**。

- 毛线没有任何收费项目，请勿相信任何与本服务器有关的现实交易行为。
  
- 管理员不会给自己加管理前后缀，如果需要请在群内召唤管理员。
  
- 请勿轻信任何牵扯个人隐私和财产的信息、请不要向任何人透露、在任何非 mojang 官方或自称 mojang 官方的地方输入自己的帐号密码或支付信息。
  
- 新玩家将赠送一些游戏币，请勿轻信恶意玩家的欺诈或诱骗高价购买非等值物品或转帐。请保持头脑清醒，明辨是非。


### 加入要求

- 您已购入 Minecraft PC 版。如果您尚未购买正版游戏，请在官方网站购买。
  
- 您已在启动器中使用正版帐号登陆。
  
- 服务器前端已安装 ViaVersion , 使您的游戏版本与服务器版本不匹配也依然可以加入游戏并在服务器间切换，但部分新版本内容及资源包内容会回退到旧版(尽管不影响功能)，为了最佳游戏体验，推荐您使用 1.21+ 版本的 MC 客户端加入游戏。
  服务器无模组要求，但推荐安装渲染优化插件（如 [Sodium](https://github.com/CaffeineMC/sodium-fabric)）。
  如希望使用辅助插件, 推荐阅读[推荐资源](wiki/resources.md)。
  
- 您已认真阅读[游戏规则](https://craft.moe/rules)。此外，推荐阅读[常见问题](kedama/faq.md)。
  
- 戳 [玩家群组](wiki/groups.md) 页面，加入合适的群。</br>
  有的 IRC、Telegram 群组和游戏聊天频道互通，请注意不要刷屏。</br>
  即便不加入任何群组，管理组依然会在毛线有重大更新或通知时，在NyaaCat论坛上发布公告
</br></br>

### 登录服务器
启动 Minecraft 客户端后，请添加服务器地址：

  `play.craft.moe`
  
即可进入游戏。

如果您在封锁高位端口的网络环境中，请尝试添加备用服务器地址：

  `play2.craft.moe`
  
请允许使用服务器资源包以正常显示家具模型(不启用资源包显示为烟火之星实体)

?> :information_source: **小贴士**

- 添加后，服务器信息右上角将可提示**连接质量（延迟）**。  
如发现延迟较高（中国大陆约为 2\~60 ms，海外约为 40\~280 ms），请从 [首页](/) 选择其它地址尝试。  
- 如出现「无法解析」、「Unknown host」等提示，请多刷新几次。  
- 在“信号”标志稳定为<span class="nw-inf-green">绿色</span>之前，不要尝试登录。如等待超过三分钟，应重启游戏。

## 新手上路

欢迎来到毛线，这里可能和其他生存服务器很不一样，但希望你玩的开心。

不论你想要做休闲的渔夫，精明的商人，勇敢的战士，建造美妙绝伦的建筑，还是想要“找出外星人、未来人、超能力者和他们一起玩”的探险家，阅读这份简单的上手指南都会让你的游戏体验倍加愉悦。
</br></br>

Be Brave, Stay Wild.
</br></br>

欢迎来到毛线物语的世界，充满未知的旅程即刻开始。

### 起点

随着一片欢迎声中，一只新鲜的你出现在这个世界。不妨向大家回打招呼体现你的礼貌和素养，简单的自我介绍则让大家更容易记住、称呼你。开始游戏之前，别忘了阅读[游戏规则](https://craft.moe/rules)。

在第一次触碰这个世界的方块时，请务必小心——毛线是全部由玩家自主建设的世界，即便是出生点也不会有额外的保护措施。毛线没有所谓的“主城”，出生点附近一般会有当前世界的状况、一些商店和 NPC 以及一些新手福利箱。如有需要，可从福利箱中适当拿取物品。

“去哪儿生存呢？”——这几乎是每只新人都会提出的问题。在毛线，玩家可以自主规划土地。顺着出发的路牌踏出探险的第一步，去周游世界，去拜访聚落，去寻找心目中的世外桃源。也许你会发现一片依山傍水的繁花森林，也许你在陡峭的悬崖下找到了温暖的炉火，又或者偶遇聚落玩家的热情难却。因此种种，你决定在这里落脚定居，建设家园。

毛线是自由的世界。大胆地去探索、去寻觅、去遇见，梦想的居所便已在途中等候。

### 资源
在毛线辽阔的土地上，未经开发的森林、矿脉由全体玩家共享。如果你居住在无人规划的区域，可以在自己的领地范围内砍伐树木或挖掘矿石。如果你加入了聚落，在聚落外与其他聚落的间隔区域中也可以砍伐树木、获取其他资源。但是为了美好世界的可持续发展，砍树务必补种，探矿务必照明。

此外，也存在名为 vx 的探索用主世界、下界和末地。在这些世界中可尽情采集资源，但是如在这些世界建设或存放资源，它们也随时会被吞噬。

一部分聚落为玩家提供[公共资源设施](kedama/public-facilities.md)，有些甚至会提供[现成的房屋](kedama/towns.md)。

TIP: 在主世界挖掘时请注意：如挖进了聚落领域范围，或者不慎挖进了他人的地下设施，请原样修补并避开该处。无力修补的，应联系相关方面或管理组协助。

### 聚落

毛线是一个尽可能做到开放和自由的社区，管理组不进行任何规划或直接协助与干涉。因此毛线服务器中目所能及的一切都由建筑与生电玩家合作完成。尽管并没有 /party 指令，这里依然有相似的玩家社团(聚落)。

如希望加入较活跃的公开聚落，目前可以考虑：**[雪落白堤]，[桃花源]**。

聚落图鉴：[聚落列表与简介](kedama/towns.md)。

如有不明之处，请联系设施或聚落的负责人，或者**加入该聚落的交流群**。  
一些聚落可能已经很久无人活跃了。这种情况下，切忌贸然入住。

又或者，向往宁静喜欢独居的你会在野外找到了一片花海。毛线世界如此之大，必会找到你所喜爱的居所。自己温馨的小窝装修完成之后，可以邀请大家来参观。当然如果手艺尚待提升，也没有关系，可以在交流中学习更多姿势～别忘了摆张床，睡一觉，设个家。


## 每个卷都要精打细算

**「卷」**是这里的硬通货；购买物品、服务等都会用到，一不小心就会花光。因此，需要先了解这里经济的[运行规则](kedama/economic.md)，思考如何平衡收支。  

**贴士：**随时使用命令 `/bal` 打开你的钱包。

要是实在止不住自己的手，两下败光了，怎么办？  
**——开店！**

 [ v6 商业街](kedama/public-facilities.md#v6世界) 中选中心仪的空铺面，立起一块写着 `[shop]` 的牌子，即刻开业！用 `/u signedit [2-4] [简介]` 吸引顾客点击！最后打上 `/hm my offer [单价]` 上架第一个商品！ 

又或者使用船新的单品买卖店 - [喵喵商城](https://community.craft.moe/d/4102/12)！

如果你有珍贵、或大热门物品待售，而自家店面不易被发现，不妨考虑 [天喵商城](tutorial/plugins/hmarket.md?id=hm "世界商店")！上天喵，秒（拍）到你说好！  
输入命令 `/hm` 即可访问天喵，百种热门商品，满足你的不时之需！

当然生意总是细水长流。如果你需要一大笔启动资金，不妨寻找任务发布栏、询问在线玩家等，认领、完成各种工程任务来赚取大笔资财。

### 你是特别的

喵毛世界中的大家都有花花绿绿的称号。别眼馋了，你也可以有！

*   更改前缀 `/u chat prefix set [前缀]` 例如 `/u chat prefix set &a&l&n&o我是前缀`
*   更改后缀 `/u chat suffix set [后缀]` 例如 `/u chat suffix set &#66ccff我是后缀`

更改前后缀需要一定的卷，详情见 [Ukit 插件指南](tutorial/plugins/commands-help.md)。

### 我担心迷路，怎么办？

在世界冒险，一张精确的地图必不可少。例如，使用小地图 Mod （如Xaero's Map），它们可以确定你的位置，协助找到回家的路。更加全能的办法，是按下F3键，游戏画面左上角将显示你的实时方位。往世界中心走总是没错的！  

最万能的办法，当属 `/espawn (回到v6重生点)` ！

当然，**你可以随时询问其他伙伴**，他们会帮助你脱离困境。


## 回馈社区

毛线是玩家为核心的社区型服务器。每一位玩家都应得到尊重，都应被包容于集体。社区的良好氛围有益于引导玩家整体、有利于社区的自愈。因此，在任何可能的情况下，踊跃提出问题、参与讨论，甚至直接参与社区建设，都在一点一滴让它变得更好。

还记得刚加入的时候免费赠送的武器道具和食物吗？希望你同样能善待每一位成员，对需要帮助的玩家伸出援助之手。社区的成长和发展，需要社区成员的不断回馈。这也是社区立足之根本。

远道而来的探险家，请用心享受这精彩充实的旅程。
