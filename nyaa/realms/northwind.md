# 北风城

-   **旧称：北方主城**
-   类型：主城，公共规划区
-   范围：(-266, -2816), (1084, -1734)
-   **名誉城主：** *KK `Kuroi_Kuroha`，浅羽家风筝 `yahagi1943`*
- **居委会主任：** *无限 `CealliumInfinity`*

北风城（简称“北城”）是最早期的工业化风格城市，拥有完善的公共基础设施。

![地图](../../assets/images/realms/Northwind-map.jpg)

## 本地交通

目前有一级铁路、二级铁路，以及市内道路，连通市中心、南郊、西郊、机场等区域。  
另有传送中心、传送门等；飞行塔暂缺。

#### 传送中心

入口位于坐标  `(348, -2345)`；  
目前设有12个传送牌，允许玩家购置空牌。

#### 下界、末地传送门

由车站广场西侧的山洞进入，左手可见下界门、右手可见末地门。  
北风城因此成为低成本快速进入末地的最佳中转地之一。

## 商业服务

#### 北风城商业大厦

坐标  `(240, -2400)`；<br />
一楼仅用于设置银行柜台。  
二楼以上，允许**以聚落为单位，占用整层（北风城使用二楼）**。如需占用，请向居委会主任团报备；应自行装修内饰，请勿修改建筑外观。各聚落玩家可于各自申请的楼层设置零售木牌。

地下有公共储物箱，可对空箱上锁占用；如果被占满，请联系_卷_ （或其他管理员）处理。

#### 北千里事务大厦

坐标  `(240, -2450)`；<br />
玩家可租用其中一层用于商业事务所办公之用。请注意，大厦内不允许建立商店。


## 公共服务

#### 北风城市政厅

坐标  `(305, -2436)`；<br />
市政厅提供各类公共服务站和官方运营物品（目前仅**飞行燃料**）的兑换/购置。

#### 初级产品集散市场

坐标  `(279, -2400)`；<br />
出售大部分原版初级建材，包括沙石料、所有原木、部分下界及海底产品等。

该处曾称为「矿物精炼中心」，在1.13.2版本及以前，玩家在此可通过消耗青金石，从原矿石中精炼出更多矿物。  
2019年9月，旧版 NPC 全部移除之后，其失去了原本功能。

#### 综合处理站

坐标 `(326, -2407)`；<br />
提供若干加工业务，包括：
- 加工**白色混凝土粉末及预拌成品**，和工业级**红沙、粘土**等副产品。
- 从TNT提取**火药**。
- 从腐肉提取蛋白质，缝制**线、羽毛**。用线也可制作羽毛。

该处曾称为「魔法分解中心」，在1.11.2版本及以前，以固定价格“回收”部分常见黑化怪战利品。  
2017年6月，旧木牌商店（SignShop）插件下线之后，其失去了原本功能。

其隔壁为发电厂。

#### 美食店（蜂蜜蛋糕房）

坐标  `(342, -2446)`；<br />
~~专门销售各种蛋糕甜品~~，拥有情侣餐桌和家庭聚会桌供玩家们约♂会。  
现不再提供蛋糕，蛋糕仅作展示之用。如果觉得饿，去找好心人家罢。

最近，名叫 Steve 的厨师收购了蛋糕房，开始销售快餐盒。

#### 城区公共仓库

坐标  `(324, -2388)`；<br />
用于短期、中期存放物品的公共仓库。  
如确认使用某空箱，请锁上箱子；将物品搬走后请将锁移除。

#### 北城南郊林场

位于第三居民区以北、水库以南；  
仿自然林场，提供**全部六种木材**，缺原木、缺树苗、缺藤蔓、缺可可豆，在此都能解决。林场内**全境覆盖充足光照**，避免怪物生成（除幻翼）。某热心市民会不定期巡山，检视并完善植被与光照。  
请于光照充足之处工作，以策安全。请自然、科学补种。

!> **当心幻翼。** 如需长时间作业，请及时睡觉，或在夜间到树荫下或隧道内避难。

## 历史
- **建城之初**，城市处于无序发展状态。
- **2015年**
  + 在以*KK* 为首的“北方建设集团”（NCG）主持下，原北方主城大规模翻修，中心城区自此定型。同时，三个居民区（Arch、第一区、第二区）及水库固定下来。
  + 同年，“北城环线高铁”建成；“北城新区”成立，接纳原来无法于北城大展拳脚的建筑师。
- **2016年**
  + 第三居民区、机场开辟，火车南站设立。
  + “环线高铁”被拆除。
  + 第一区东部被重置<sup>（需要证明）</sup>。
  + “北城新区”更名为[新北市](nyaa/realms/newnorth)，脱离北城管辖。
  + “飞行燃料”上线，北城市政厅承接部分兑换配方。
- **2017年**
  + 第一区东部与第三区扩充自建地块。
  + 6月25日，因插件调整，魔法分解中心失去功能。
  + 七月，经*浅羽家风筝* 授权，*无限* 在南郊荒山上建立并经营林场。
  + 约莫八月，*浅羽家风筝* 离开喵窝，北城发展几乎中断。
- **2018年**
  + 5月，*无极限拉玛牧业有限公司* 于北城成立，开始有限接管城市建设。
- **2019年**
  + 3月至7月，*无极限拉玛公司* 试发行杂志[《北城拉玛电台·新闻月刊》](https://bbs.nyaa.cat/d/1388-2019-3-5-2019-7-20/14)，总计发行五期。
    * 杂志副本现可于**北风城图书馆**查阅。
  + 9月，因内容调整，精炼中心及图书馆失去功能。
  + 9月底，*无极限拉玛公司* 宣布改组，成立[集团公司](https://bbs.nyaa.cat/d/1293/17)。居委会成立前，其仍有限维护城市基建。
  + 10月，城市不再以“主城”自居；改名“北风城”，沿用至今。
  + 居民委员会于12月1日成立，*无限* 任主任；并代管新北市街道。
- **2020年**
  + 3月1日，原精炼中心、分解中心相继改制，接收柚木小镇的工业功能。

## 你知道吗？

-   北风城曾经存在环城铁路线、地铁系统。目前仍可于西北郊、北风城西站，发现环城线之遗迹（如支柱、月台）；中心城区广场、图书馆等处可见地铁站遗迹。
- 在北风城东郊近海上，有一片巨大的、功能未知的浮空方块“阵列”，在原版地图上，其形如显像管测试画面。因高度位于世界最高`Y=255`，其被VoxelMap显示为“一团黑色矩形”；而在JourneyMap则显示正常。
- 在第一区某民居，有全窝罕见的**僵尸马**。
- 在城区西北部的沼泽区域（包括商业大厦旁的树下），有机会生成史莱姆。
